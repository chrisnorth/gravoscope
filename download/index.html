Gravoscope tilesets

Below are the various tile-sets used in Gravoscope. If you would like to run a local version, without requiring access to the internet, then you will need to download these. The source code can be downloaded from the Github respoitory.

Each tile-set contains over 5000 images, archived and compressed into both tarballs (.tar.gz) and zip files. While the compressed archives are not too large (typically 10-50 MB), the uncompressed files can take up rather a large amount of space and take considerable time to extract.

Note that while the owners of the original images have given us permission to use the images in the context of Chromoscope, we do not hold the copyright to them. As with Chromoscope itself, the tilesets are free for educational and non-commercial use. For other uses, please contact the original owners.

Generating Tilesets

Tilesets can be generated using the Perl script cutter.pl (written by Stuart Lowe) or the Java utiliy GMapImageCutter (originally from here).
